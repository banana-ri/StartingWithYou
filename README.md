# 👕 시작의 아이

**시작의 아이**는 사용자의 옷장 관리와 유니티(Unity) 기반 가상 피팅 서비스를 목표로 하는 안드로이드 애플리케이션 프로젝트입니다.

---

## 🚀 현재 개발 진행 상황 (Status Report)

팀 프로젝트의 효율적인 협업을 위해 현재까지의 작업물과 개발 단계를 공유합니다.

### 🛠 주요 기능 및 작업 내용

#### 1. 나의 옷장 (My Closet) 🚧
* **상태:** 세부 열람 기능 로직 구현 완료
* **내용:** 옷장에 등록된 각 아이템의 상세 정보를 확인할 수 있는 기능을 개발 중입니다.
* **주의사항:** **실제 기기 테스트 전** 단계이므로, 데이터 바인딩 시 예외 상황이 발생할 수 있습니다.

#### 2. 가상 피팅 (Virtual Fitting) 인터페이스 준비 🦴
* **상태:** 화면 구조 및 진입점 생성 완료
* **위치:** `try_page` 및 `tryfragment` 관련 파일
* **상세:** 추후 **유니티 연동 로직이 탑재될 자리**를 프래그먼트 및 페이지 구조로 미리 설계해 두었습니다.
* **주의사항:** 현재 버전에는 **실제 유니티 연동 로직 및 데이터는 포함되어 있지 않으며**, 화면 전환 구조만 잡혀있는 상태입니다.

#### 3. UI/UX 디자인 수정 예정 🎨
* **상태:** 프로토타입 레이아웃 적용
* **내용:** 현재 적용된 레이아웃은 임시 디자인으로, 차후 테스트 기기에 맞추어 수정이 필요합니다. 
---

## 📂 주요 파일 구조
| 기능 | 경로 | 비고 |
|:---:|:---|:---|
| **가상 피팅 페이지** | `app/src/main/.../try_page` | 가상 피팅 진입 인터페이스 |
| **가상 피팅 뷰** | `app/src/main/.../tryfragment` | Unity 연동 예정 위치 |

## 💬 팀원 전달 사항
1. **빌드 환경:** 최신 Gradle(8.x) 및 KSP 설정을 마쳤습니다. 프로젝트 오픈 후 `Sync Project`를 반드시 진행해 주세요.
2. **Unity 작업:** 가상 피팅 관련 작업은 `tryfragment`를 중심으로 결합될 예정이니 해당 파일의 구조를 참고 부탁드립니다. 새 브랜치를 만들어 작업해 주세요.
3. **피드백:** 기능 동작 오류나 레이아웃 개선에 대한 의견은 언제든 공유해 주세요!
